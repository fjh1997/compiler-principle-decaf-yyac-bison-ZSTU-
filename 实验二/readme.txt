 PP2中是实验的第二部分－－语法分析的相关文件。里面有
    三个文件夹：documents、StarterFiles和samples。

	documents中是语法分析部分的文档，其中:
	“pp2实验说明.doc”是语法分析部分的总体说明，请大家在
    做实验前仔细阅读。
	“pp2实验说明(eng).doc”是国外大学编译原理课件中相应的
    文档，其主要内容“pp2实验说明.doc”中已经有了，但还有一些
    实现方法的指导和可选的扩展内容，对做好实验很有帮助。
	"bison.info"是bison工具的使用说明和帮助文档。bison是一
    个自动构造语法分析程序的工具。在PP2中，我们就是要 
   使用bison来生成语法分析程序，所以这份文档非常有用。不过由于它
    比较长，大家可以先看一下教材附录C中YACC的使用方法部分，然后再
    浏览一下这份文档。编码时遇到问题再到文档中来查。
	"YACC的使用方法和PP2实验说明.ppt"是我在课堂上给大家布置
    实验时的演示文档。
	StarterFiles是我们为大家提供的程序框架和工具。其中主要文
   件的说明在“pp2实验说明.doc”中已经有了，这里不再赘述。大家只
   要按照说明修改相应的文件就可以完成编码工作了。
	samples中是我们为大家提供的实例，包括对应的Decaf源文件和
    相应的正确的语法分析输出结果，供大家测试程序。具体的使用方法
    在“pp2实验说明.doc”中也已经有了。
