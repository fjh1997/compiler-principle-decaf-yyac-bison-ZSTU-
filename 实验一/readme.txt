    ---------------------------------------
    编译原理实验的内容是实现一个Decaf语言的编译器.这里是一些有用
    的文档和代码,具体内容说明如下:


    1.Documents中是与整个实验有关的说明和帮助文档。
     "Decaf_language.rtf"是Decaf语言的说明.请大家在各部分实验时阅
     读相关内容.
	
    2.PP1中是实验的第一部分－－词法分析的相关文件。里面有三个文件
    夹：documents、StarterFiles和samples。documents中是词法分析部
    分的文档，其中:

       “实验一说明.doc”是词法分析部分的总体说明，请大家在做实
    验前仔细阅读。
       “实验一说明(eng).doc”是英文的文档，其主要内容“实验一说明.doc”中已经有了，但还有一些实现方法的指导和可选的扩
    展内容，对做好实验很有帮助，请大家着重看一下“scanner 
    implementation”和“optional extensions”两部分。
	

        StarterFiles是我们为大家提供的程序框架和工具。其中主要文
   件的说明在“实验一说明.doc”中已经有了，这里不再赘述。大家只
   要按照说明修改相应的文件就可以完成编码工作了。
	samples中是我们为大家提供的实例，包括对应的Decaf源文件和
    相应的正确的文法分析输出结果，供大家测试程序。具体的使用方法
    在“pp1实验说明.doc”中也已经有了。

    如有问题、建议和意见，欢迎大家随时与我联系。
